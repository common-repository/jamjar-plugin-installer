Jamjar plugin installer allows you to install multiple plugins in a single go.

== Description ==
Jamjar Plugin Installer is a highly compatible wordpress plugin with a great user friendly interface,it allows user to install multiple plugins with a single click.All you have to do is to drag and drop your plugins into the jamjar plugin installer.

<strong>MULTISITE SUPPORTED</strong>
Have you ever noticed that the option to add new plugins is filtered in your multisites? and the only way is to install it from the network dashboard ,with JamJar you can add plugins directly into your sites without the help of network dashboard.

== Installation ==
Manual installation
1. Download and unzip the jamjar-plugin-installer.zip
2. Upload unzipped files into the `/wp-content/plugins/` directory trough web file manager or ftp/sftp.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Find the configuration menu under the "Plugins" Menu.
